generator-boomerang

generator-boomerang is a Yeoman generator for GDG Boomerang.

  • GDG Boomerang makes it really easy to get a website up and running for your GDG chapter.
  • Yeoman turbo charges your web development with a set of tools to help in your modern workflow.

How do I use generator-boomerang?

You'll first have to setup yeoman. If you have Node.js installed it is as easy as running ...

nmp install -g yo

If you need help setting up node or yeoman check out their getting started guide

I have yeoman ready, now what?

If you have your environment all setup and ready to go you can run

npm install -g generator-boomerang  # install generator
yo boomerang                        # scaffold out a boomerang site

Just a few question.

The generator will guide you through setup with a few questions. It's probably a good idea to have the answer to these before you begin.

What is your GDG Chapter name?         # This is what will show up in the header of navigation
What is your GDG Chapter Google+ ID?   # the numerical id of your GDG Google+ chapter page
What is your Google API Key?           # get this from https://code.google.com/apis/console
What is your Picasa Web Album ID?      # the numerical id of photo album that belongs to your Chapters +Page
                                       # please note that it can NOT be related to an event album

After you have answered all the questions, yo will build out your project. from there you can just run

grunt server

grunt will fire up a server and display your app.

Is it really that easy?

Yes.

How do I get it on the web?

Run

grunt build

This will build out your project, minify js/css and test it for errors. In your project folder you will get a dist folder. In there is the contents that you need to publish on your remote server.

Your done.
